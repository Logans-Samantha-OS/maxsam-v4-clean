# HttpClient MCP Server

Generic REST API wrapper

## Installation

```bash
cd C:\Users\MrTin\Downloads\MaxSam-V4\mcp-servers\http-client-mcp

```

## Configuration

Add to `claude_desktop_config.json`:

```json
{
  "http-client": {
    "command": "node",
    "args": ["C:\Users\MrTin\Downloads\MaxSam-V4\mcp-servers\http-client-mcp\index.js"],
    "env": {
      
    }
  }
}
```
