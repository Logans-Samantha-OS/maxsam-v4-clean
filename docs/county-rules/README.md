## Key Information (All Counties)

### Universal Rules (Texas Tax Code Section 34.04)

| Item | Details |
|------|---------|
| **Deadline** | 2 years from tax foreclosure sale date |
| **Minimum Claim** | $25 or more |
| **Process** | File petition with county district court |
| **Required** | Court order signed by presiding judge |

### Who Can Claim

1. Former property owners (deed recorded BEFORE judgment date)
2. Lienholders with recorded financial interest
3. Heirs of former owner (with proof of relationship)

### Distribution Priority

1. Tax sale purchaser (if sale voided)
2. Taxing units for subsequent taxes
3. Lienholders for amounts due
4. Taxing units for judgment amounts
5. Former owners
